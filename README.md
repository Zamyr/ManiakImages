# Maniak Images Project

Creation of the project to see images using maniak's api.

## Installation

- Clone the project
- When you have the project you need to run this command

```bash
npm install
```

- if you are in Mac Os, run this command

```bash
cd ios && pod install
```

- Run the project

Android
```bash
npm run android
```

IOS
```bash
npm run ios
```
